# Цель проекта
Изучить развитие игровой индустрии с 2000 по 2013 год, выявить наиболее популярные игровые консоли, а также категоризировать экспертные и пользовательские оценки.

# Описание проекта
В процессе исследования были собраны данные об играх, платформе и годе выпусках, жанрах, продажах в разных регионах, пользовательских и экспертных оценках. Данные были отфильтрованы для исследования игр только за 2000-2013 гг. Пользовательские и экспертные оценки были разделены на категории: "низкая оценка", "средняя оценка", "высокая оценка". Был выведен топ-7 платформ по количеству вышедших игр.

# Выводы исследования
1. В период 2000-2013 гг. больше всего игр выходило для консоли PS2.
2. Подавляющее большинство оцененных игр подходит под категорию "средняя оценка".
