# Цель проекта
Готовый анализ объявлений о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, чтобы найти самые перспективные сегменты недвижимости.
# Описание проекта
Было необходимо:
- Выявить наиболее перспективные сегменты рынка недвижимости Санкт-Петербурга и городов Ленинградской области для формирования эффективной бизнес-стратегии. В качестве показателя привлекательности использовать продолжительность размещения объявлений.
- Проанализировать сезонность рынка недвижимости Санкт-Петербурга и городов Ленинградской области с целью определения периодов повышенной активности продавцов и покупателей. Полученные выводы позволят оптимально планировать маркетинговые активности и выбирать наиболее подходящее время для выхода на рынок.

# Инструменты проекта
